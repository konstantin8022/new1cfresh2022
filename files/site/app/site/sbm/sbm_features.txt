<h4>Ведите в&nbsp;одной базе учет нескольких организаций.</h4>

<p>Как независимых, так и&nbsp;работающих в&nbsp;рамках одного бизнеса. Даже при изменении масштабов, структуры бизнеса или подходов к&nbsp;организации деятельности предприятия вы легко продолжите вести учет без больших затрат времени и&nbsp;денег.</p>

<h4>Планируйте, оперативно учитывайте и&nbsp;эффективно управляйте различными участками учета на предприятии:</h4>

<ul>
	<li><b>Продажи.</b> Ведите учет торговых операций: формируйте заказы покупателей, фиксируйте продажи товаров, в&nbsp;т.ч. розничные.</li>
	<li><b>Запасы и&nbsp;склад.</b> Ведите учет запасов, рассчитывайте потребность в&nbsp;них.</li>
	<li><b>Снабжение и&nbsp;закупки.</b> Отслеживайте цены поставщиков и&nbsp;выбирайте лучшие условия закупки, формируйте заказы поставщикам, фиксируйте поступления товаров.</li>
	<li><b>Сервис.</b> Учитывайте оказание услуг и&nbsp;подрядных (проектных) работ: формируйте заказы и&nbsp;заказ-наряды, фиксируйте выполненные работы.</li>
	<li><b>Производство.</b> Планируйте загрузку ресурсов предприятия, ведите спецификации, вносите календарный график производства, регистрируйте заказы на производство, выпуск продукции, ведите учет производственных операций, рассчитывайте потребности в&nbsp;материалах и&nbsp;фактическую себестоимость продукции с&nbsp;учетом прямых и&nbsp;косвенных затрат.</li>
	<li><b>Денежные средства.</b> Учитывайте движение денежных средств на расчетных счетах и&nbsp;в&nbsp;кассах, ведите платежный календарь и&nbsp;определяйте приоритеты платежей. Используйте обмен данными с&nbsp;программами типа &laquo;Клиент банка&raquo;.</li>
	<li><b>Взаиморасчеты.</b> Ведите расчеты с&nbsp;контрагентами, сотрудниками в&nbsp;разрезе договоров, заказов, документов.</li>
	<li><b>Зарплата и&nbsp;персонал.</b> Храните данные о&nbsp;сотрудниках, составьте штатное расписание, регистрируйте кадровые документы по приему, перемещению и&nbsp;увольнению сотрудников. Планируйте нагрузку и&nbsp;оценивайте эффективность работы персонала, ведите графики работы и&nbsp;табельный учет рабочего времени, контролируйте выполнение нарядов и&nbsp;заданий сотрудниками. Рассчитывайте управленческую зарплату на основе настраиваемых показателей &ndash; как при повременной, так и&nbsp;при сдельной системе оплаты труда.</li>
	<li><b>Имущество.</b> Учитывайте материальное и&nbsp;нематериальное имущество с&nbsp;длительным сроком использования, начисляйте амортизацию, проводите инвентаризацию.</li>
</ul>

<h4>Управляйте взаимоотношениями с&nbsp;клиентами.</h4>

<p>Приложение позволяет вести базу клиентов, историю взаимоотношений с&nbsp;ними, планировать звонки, встречи, переговоры, учитывать взаимные обязательства.</p>

<h4>Планируйте свою деятельность.</h4>

<p>Регистрируйте планируемые хозяйственные операции и&nbsp;намеченные события: заказы поставщикам, заказы покупателей, состояние заказов, задания сотрудникам, распределение загрузки ресурсов предприятия, планы-графики выполнения работ и&nbsp;оказания услуг, планы-графики производства, планы закупок и&nbsp;продаж и&nbsp;многое другое.</p>

<h4>Используйте электронный документооборот с&nbsp;клиентами и&nbsp;поставщиками.</h4>

<p>В&nbsp;приложение встроен сервис для обмена юридически значимыми документами (счетами-фактурами, договорами, накладными и&nbsp;пр.) в&nbsp;электронном виде &ndash; &laquo;1С-Такском&raquo;.</p>

<p><strong>Обрабатывайте заказы покупателей с&nbsp;сайта интернет-магазина.</strong><br />
Вы можете настроить обмен данными с&nbsp;интернет-магазином, при котором на сайт выгружается список товаров, а&nbsp;в&nbsp;учетную систему &ndash; заказы, оформленные в&nbsp;интернет-магазине. Это позволит вам:</p>

<ul>
	<li>создать и&nbsp;вести единую базу заказов независимо от источника их поступления (и&nbsp;с&nbsp;сайта, и&nbsp;оформленные в&nbsp;магазине, и&nbsp;по телефону или по электронной почте и&nbsp;пр.);</li>
	<li>отображать на сайте только актуальную информацию о&nbsp;товарах, их наличии и&nbsp;стоимости товаров.</li>
</ul>

<h4>Бизнес всегда под контролем!</h4>

<p>Организуйте финансовое планирование и&nbsp;бюджетирование для эффективного распределения и&nbsp;использования ресурсов. Регистрируйте плановые данные и&nbsp;формируйте:</p>

<ul>
	<li>прогнозный финансовый результат,</li>
	<li>бюджет прибылей и&nbsp;убытков,</li>
	<li>бюджет движения денежных средств.</li>
</ul>

<p>Для анализа и&nbsp;контроля состояния дел формируйте различные отчеты, которые представляют информацию быстро, в&nbsp;удобной форме, с&nbsp;требуемой детализацией:</p>

<ul>
	<li>Управленческий баланс, отчеты о&nbsp;прибылях и&nbsp;убытках, о&nbsp;движении денежных средств предназначены для общего анализа ситуации на предприятии.</li>
	<li>Оборотно-сальдовая ведомость позволяет получить финансовую отчетность за произвольный период времени.</li>
	<li>&laquo;Монитор руководителя&raquo; отражает основные сводные показатели деятельности предприятия для анализа и&nbsp;своевременного выявления точек роста и&nbsp;негативных динамик.</li>
</ul>

Вы можете сдавать отчетность через Интернет из &laquo;1С:Бухгалтерии 8&raquo; с&nbsp;помощью встроенного сервиса &laquo;1С-Отчетность&raquo;. Вариант обмена данными и&nbsp;время его выполнения вы подберете исходя из потребностей вашей деятельности.</p>
